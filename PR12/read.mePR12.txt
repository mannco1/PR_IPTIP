Описание классов
Класс type4 решает уравнения вида Ax^2 + C = 0, где A и C - заданные коэффициенты.
Класс type5 решает уравнения вида Bx + C = 0, где B и C - заданные коэффициенты.
Класс type6 решает уравнения вида Ax^2 + Bx + C = 0, где A, B и C - заданные коэффициенты.
Класс type8 решает уравнения вида Ax^2 + Bx + C = 0, где A и B - заданные коэффициенты.

Функционал классов
Каждый класс имеет следующие методы:

Конструктор, который принимает коэффициенты уравнения и сохраняет их в полях класса.
Метод Get_answer, который находит точное решение уравнения, если оно существует, и выводит его на экран. Если решения нет, то выводится соответствующее сообщение.
Метод show, который выводит уравнение на экран в читаемом виде.
Метод Get_answer_podbor, который находит приближенное решение уравнения методом подбора из заданного интервала и выводит его на экран.
