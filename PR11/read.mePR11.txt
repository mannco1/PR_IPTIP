В коде используются два класса: type7 и type8.

Класс type7 предназначен для решения уравнений вида Bx = 0, где B - ненулевое число. Конструктор этого класса принимает один параметр b1, который является коэффициентом B. Метод Get_answer() выводит на экран единственный корень уравнения x = 0. Метод show() выводит на экран уравнение в виде B*x = 0.
Класс type8 предназначен для решения уравнений вида Ax^2 + Bx = 0, где A и B - ненулевые числа. Конструктор этого класса принимает два параметра a1 и b1, которые являются коэффициентами A и B соответственно. Метод Get_answer() выводит на экран два корня уравнения: x1 = 0 и x2 = -B/A. Метод show() выводит на экран уравнение в виде Ax^2 + Bx = 0.
